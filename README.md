# Dislack

### What is Dislack?

Dislack is a combinaton of Discord and Slack

### What platforms does Dislack support?

Dislack supports the following platforms:

* Windows (Compose for Desktop)
* Linux (Compose for Desktop)
* macOS (Compose for Desktop)
* Android (Compose for Android)
* (Soon) iOS (SwiftUI)
* (Soon) Web (Compose for Web)
